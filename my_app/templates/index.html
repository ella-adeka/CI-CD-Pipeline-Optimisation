CI/CD Pipeline Optimisation

This project focuses on automating the Continuous Integration and Continuous Deployment (CI/CD) of a web application using Jenkins and Docker. The goal is to streamline the development process, reduce manual intervention, and ensure efficient and reliable application delivery.

  2. Code Writing (Visual Studio Code)
  3. Web Application Framework (Python (Flask))
  4. Version Control (Git)
  5. Build Automation (Jenkins)
  6. Containerisation (Docker)
  7. Infrastructure as Code (IaC) (Terraform)
  8. Integration Testing Framework (pytest)
  9. Cloud Service (Amazon Web Services (AWS))
  10. Monitoring & Logging (Prometheus and Grafana)

AWS Services

  1. Amazon EC2 - Computing Resources
  2. Amazon S3 - Storage
  3. Elastic Load Balancer - Distributing incoming traffic
  4. Autoscaling - Auto increase number of instances running application
